# API请求文档

## 接口列表

### 支付请求:
地址：`/payment/pay`  
方法：`POST`  
参数：  

| 参数名 | 类型 | 说明 | 是否必选 | 备注 |
| --- | --- | --- | --- | --- |
| version | 字符 | 版本号 | 是 | 例:V1.0(默认) |
| merchant_id | 字符 | 商户号 | 是 | 系统后台自动分配商户号 |
| data | 字符 | 加密后的数据 | 是 | 采用RSA加密，填充方式：PKCS1 |

data参数解密说明:
  
| 参数名 | 类型 | 说明 | 是否必选 | 备注 |
| --- | --- | --- | --- | --- |
| version | 字符 | 版本号 | 是 | 例:V1.0(默认) |
| merchant_id | 字符 | 商户号 | 是 | 系统后台自动分配商户号 |
| data | 字符 | 加密后的数据 | 是 | 采用RSA加密，填充方式：PKCS1 |
